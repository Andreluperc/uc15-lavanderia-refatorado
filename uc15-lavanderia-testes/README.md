# uc15-lavanderia-testes (JUnit)

Projeto Maven de testes unitarios (JUnit 5) para apoiar o PI.

## Como executar os testes

No terminal, dentro desta pasta:

```bash
mvn test
```

Ou pelo NetBeans:
- Clique com botao direito no projeto -> Test

## O que esta sendo testado

- `CpfUtils`: normalizacao e validacao basica de CPF (regra simples, sem acesso ao banco).
